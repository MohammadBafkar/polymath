# polymath-performance

Performance craft for the Polymath marketplace.

## What it ships

- Skills: `perf-budget`, `backend-tail-latency`, `caching-tradeoffs`, `design-cache-layer`, `audit-redis-config`.
- Commands: `/design-cache` (alias over `design-cache-layer`), `/audit-redis-config` (alias over `audit-redis-config`).

## Installation

```bash
claude plugin install polymath-performance@polymath
```

## Dependencies

- `polymath-core`
