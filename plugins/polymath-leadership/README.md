# polymath-leadership

Leadership craft for the Polymath marketplace.

## What it ships

- Skills: `one-on-one-prep`, `perf-review`, `okr-setting`.
- Templates: `1-on-1.md`, `Perf-review.md`, `OKR.md`.

## Installation

```bash
claude plugin install polymath-leadership@polymath
```

## Dependencies

- `polymath-core`
