# polymath-lang-kotlin

Kotlin craft for the Polymath marketplace.

## What it ships

- Skills:
  - `write-kotlin-test` — JUnit 5 / Kotest + MockK + `kotlinx.coroutines.test.runTest`; backtick-named test functions.
  - `audit-gradle-kts` — Kotlin / JVM target alignment, plugin pinning, BOM use, multi-module hygiene, reachable vulnerabilities.
- Commands: `/polymath-lang-kotlin:test`, `/polymath-lang-kotlin:audit`.

## Installation

```bash
claude plugin install polymath-lang-kotlin@polymath
```

## Dependencies

- `polymath-core`, `polymath-engineering`
