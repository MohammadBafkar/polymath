# polymath-lang-ruby

Ruby craft for the Polymath marketplace.

## What it ships

- Skills:
  - `write-rspec-test` — describe/context/it nesting, `let`/`subject` fixtures, `instance_double` over bare `double`, `shared_examples` for cross-class behavior.
  - `audit-gemfile` — Ruby version, pin discipline, bundler-audit findings, sources, group separation.
- Commands: `/polymath-lang-ruby:test`, `/polymath-lang-ruby:audit`.

## Installation

```bash
claude plugin install polymath-lang-ruby@polymath
```

## Dependencies

- `polymath-core`, `polymath-engineering`
