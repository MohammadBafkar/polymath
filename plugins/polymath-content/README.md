# polymath-content

Customer-facing content craft for the Polymath marketplace. Distinct from `polymath-release` (engineering-facing) and `polymath-writing` (internal docs).

## What it ships

- Skills: `write-release-notes`, `write-sunset-notice`, `write-advisory`.
- Templates: `Sunset-notice.md`, `Advisory.md` (materialized from `shared/templates/`).

## Installation

```bash
claude plugin install polymath-content@polymath
```

## Dependencies

- `polymath-core`
