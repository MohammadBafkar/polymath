# polymath-lang-swift

Swift craft for the Polymath marketplace.

## What it ships

- Skills:
  - `write-xctest` — XCTest or Swift Testing depending on project; async/await first, `XCTestExpectation` only when legacy.
  - `audit-package-swift` — tools version, platforms, dependency pinning, dev-vs-runtime separation.
- Commands: `/polymath-lang-swift:test`, `/polymath-lang-swift:audit`.

## Installation

```bash
claude plugin install polymath-lang-swift@polymath
```

## Dependencies

- `polymath-core`, `polymath-engineering`
