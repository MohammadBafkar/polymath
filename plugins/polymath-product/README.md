# polymath-product

Product craft for the Polymath marketplace. Drafts PRDs, refines acceptance criteria, and decomposes epics into vertical slices.

## What it ships

- Skills: `prd`, `acceptance-criteria`, `decompose-epic`.
- Templates: `PRD.md`, `User-story-map.md` (materialized from `shared/templates/`).

## Installation

```bash
claude plugin install polymath-product@polymath
```

## Dependencies

- `polymath-core`
