# polymath-prioritize

Prioritization craft: rank a backlog with an explicit, inspectable method and cut a Now/Next/Later roadmap — the bridge between thinking/planning and execution.

## What it ships

- Skills: `prioritize` — rank a backlog or feature set with RICE, WSJF/cost-of-delay, MoSCoW, Kano, or value-vs-effort, then cut Now/Next/Later. Exposes the value/effort/confidence inputs behind every score.

## Why it exists

Prioritization is one of the most common product/eng-lead requests and had no first-class home in the marketplace — `polymath-planning` estimates and breaks down work, `polymath-product` decomposes epics, but nothing *ranked* a set of candidates with a defensible method. `prioritize` fills that gap and consumes those skills' outputs rather than duplicating them.

## Installation

```bash
claude plugin install polymath-prioritize@polymath
```

## Dependencies
