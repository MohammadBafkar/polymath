# polymath-lang-dotnet

.NET craft for the Polymath marketplace.

## What it ships

- Skills: `write-xunit-test`, `audit-csproj-modernization`, `propose-nullable-references`.
- Commands: `/xunit`, `/csproj-audit`, `/nullable`.

## Installation

```bash
claude plugin install polymath-lang-dotnet@polymath
```

## Dependencies

- `polymath-core`
- `polymath-engineering`
