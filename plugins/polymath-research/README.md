# polymath-research

Research craft for the Polymath marketplace.

## What it ships

- Skills: `interview-guide`, `persona`, `customer-journey`.
- Agents: `research-scout` for fresh-context primary-source evidence gathering.
- Templates: `Interview-guide.md`, `Persona.md`, `Customer-journey.md`.

## Installation

```bash
claude plugin install polymath-research@polymath
```

## Dependencies

- `polymath-core`
