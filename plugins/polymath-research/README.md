# polymath-research

Research craft for the Polymath marketplace.

## What it ships

- Skills: `interview-guide`, `persona`, `customer-journey`.
- Templates: `Interview-guide.md`, `Persona.md`, `Customer-journey.md` (materialized from `shared/templates/`).

## Installation

```bash
claude plugin install polymath-research@polymath
```

## Dependencies

- `polymath-core`
