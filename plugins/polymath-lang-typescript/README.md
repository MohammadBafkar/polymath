# polymath-lang-typescript

TypeScript craft for the Polymath marketplace.

## What it ships

- Skills: `write-vitest-test`, `lint-with-biome`, `migrate-ts-version`.
- Commands: `/vitest`, `/biome`, `/ts-migrate`.

## Installation

```bash
claude plugin install polymath-lang-typescript@polymath
```

## Dependencies

- `polymath-core`
- `polymath-engineering`
