# Changelog — polymath-thinking

## [Unreleased]

### Added

- `problem-framing` skill — frame the problem before any solution (symptom vs
  root need, who/what/why, constraints, premature-solution bias) into a
  handoff-ready problem statement.
- `first-principles` skill — strip a problem to fundamentals, tag inherited
  assumptions vs laws, and rebuild the approach from bedrock.
- `deliberate` skill and `/deliberate` command for an iterative
  observe-frame-options-tradeoffs-risks-revise loop.

## [0.1.0]

### Added

- Initial v0.1 components: `brainstorm`, `5-whys`, `pre-mortem`, `red-team` skills.
