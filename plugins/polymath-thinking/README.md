# polymath-thinking

Thinking craft for the Polymath marketplace. Divergence, root-cause, prospective risk, adversarial challenge.

## What it ships

- Skills: `brainstorm`, `deliberate`, `5-whys`, `pre-mortem`, `red-team`, `problem-framing`, `first-principles`.
- Commands: `/deliberate`.
- Agents: `architecture-critic` for fresh-context ADR/design challenge.

## Installation

```bash
claude plugin install polymath-thinking@polymath
```

## Dependencies

- `polymath-core`
