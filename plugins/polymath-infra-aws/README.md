# polymath-infra-aws

AWS write-side infrastructure craft for the Polymath marketplace.

## What it ships

- Skills: `design-aws-pattern` — primitive choice (compute / storage / database / networking) with quantitative flip conditions.
- Command: `/polymath-infra-aws:design`.

## Pairs with

- `polymath-connector-aws` — read-side resource inspection.
- `polymath-connector-terraform` — review plan output before apply.

## Installation

```bash
claude plugin install polymath-infra-aws@polymath
```

## Dependencies

- `polymath-core`
