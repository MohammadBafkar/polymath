# polymath-decisions

Decision craft for the Polymath marketplace.

## What it ships

- Skills: `daci`, `tradeoff-matrix`, `cynefin-frame`.
- Templates: `DACI-decision.md`, `Tradeoff-matrix.md` (materialized from `shared/templates/`).

ADRs live in `polymath-writing:adr` (Nygard format with frontmatter schema validation). Pre-mortem decisions live in `polymath-thinking:pre-mortem`.

## Installation

```bash
claude plugin install polymath-decisions@polymath
```

## Dependencies

- `polymath-core`
