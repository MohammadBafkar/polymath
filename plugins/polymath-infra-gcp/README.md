# polymath-infra-gcp

GCP write-side infrastructure craft for the Polymath marketplace.

## What it ships

- Skills: `design-gcp-pattern` — compute / database / storage / messaging / networking choice with cost driver + ops shape + quantitative flip conditions.
- Command: `/polymath-infra-gcp:design`.

## Pairs with

- `polymath-connector-gcp` — read-side resource inspection.
- `polymath-connector-terraform` — review plan output before apply.

## Installation

```bash
claude plugin install polymath-infra-gcp@polymath
```
