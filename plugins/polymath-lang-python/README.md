# polymath-lang-python

Python craft for the Polymath marketplace.

## What it ships

- Skills: `write-pytest-test`, `lint-with-ruff`, `propose-type-annotations`.
- Commands: `/pytest`, `/ruff`, `/types-py`.

## Installation

```bash
claude plugin install polymath-lang-python@polymath
```

## Dependencies

- `polymath-core`
- `polymath-engineering`
