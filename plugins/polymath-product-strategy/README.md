# polymath-product-strategy

Product-strategy craft: the where-to-play / how-to-win choices above a feature spec — made explicit and falsifiable.

## What it ships

- Skills: `product-strategy` — vision, ICP/segmentation, positioning + messaging, pricing/packaging, moat, and GTM launch motion, each framed as a choice with the alternative rejected and a cheap test for its key bet.

## Why it exists

The audit found product-strategy/GTM tagged only by mis-attributed skills — no vision, positioning, pricing, or GTM home. This plugin adds the strategy/market layer above `polymath-product:prd` (the feature spec) and `polymath-communication:six-pager`/`exec-brief` (the write-ups).

## Installation

```bash
claude plugin install polymath-product-strategy@polymath
```

## Dependencies

- `polymath-core`
