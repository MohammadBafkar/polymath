# /new-plugin

Scaffold a new plugin via `tools/new-plugin.sh <name>`. Pass the bare name (the script prepends `polymath-`). After scaffolding, register the plugin in `.claude-plugin/marketplace.json` and run `polymath-author:validate-plugin`.
