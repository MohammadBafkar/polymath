# polymath-frontend

Frontend craft for the Polymath marketplace. Framework-agnostic — patterns apply to React, Vue, Svelte, Solid, vanilla.

## What it ships

- Skills: `component-design`, `web-vitals-budget`, `bundle-analyze`, `i18n-audit`.

## Installation

```bash
claude plugin install polymath-frontend@polymath
```

## Dependencies

- `polymath-core`
- `polymath-engineering`
