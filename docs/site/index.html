Start with a journey

End-to-end SDLC arcs that compose plugins for you. Run with /polymath-flows:run-workflow <name>, or describe the task and let polymath-core:route propose one.

JourneyWhen to run it
activateProjectSet up Polymath: project.yaml + onboarding notes.
bugTriageRoot-cause a reported bug and plan the fix.
bumpDependencyUpgrade one dependency safely and fix call-sites.
decideUnderAmbiguityFrame a hard decision, govern it, record an ADR.
deliberationLoopPressure-test a plan and emit a revised one.
deprecationToRemovalMulti-quarter, usage-gated capability removal. Not the simple two-stage notice-then-remove — use sunsetCapability for that.
designSystemDesign a system: arch doc + threat model + ADR.
estimateAndPlanScope a clear goal — work breakdown, estimate, plan.
experimentToGAA/B + flag rollout from pre-registration to GA.
featureFromIdeaDiscovery (interviews/persona) then ship a feature.
fuzzyGoalToPlanTurn a vague goal into an estimated, mapped plan.
incidentRetroToActionsTurn a postmortem into filed action tickets.
migrateLanguageVersionPhased migration to a new language/runtime version.
perfRegressionFix a perf regression and verify it recovered.
prdToShipShip from an existing PRD — acceptance through PR draft.
progressiveRolloutFlag rollout with SLO health gates (no A/B).
refactorWithSafetyPin behavior in tests, then refactor safely.
requirementsToBacklogPRD to a decomposed, groomed, ready backlog.
respondToIncidentDrive a live incident end-to-end to postmortem.
reviewPRMulti-critic review of a PR/diff + synthesis.
reviewPlanCritique an existing plan; findings only, no rewrite.
rootCauseAnalysisDrive to a system root cause; write an RCA.
securityFindingFix a security finding end-to-end and verify.
shipFeatureShip a small feature from PRD to PR draft.
sunsetCapabilityTwo-stage sunset: notice + deprecate, remove later. Not usage-gated — use deprecationToRemoval for multi-quarter, usage-gated removal.
weeklyReleaseTrainCut a release: notes, verify, tag PR.
